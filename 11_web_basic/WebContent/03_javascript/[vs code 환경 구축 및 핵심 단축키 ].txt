[ visual studio code 작업 환경 구축하기 ]

korean language pack : 한글 설치

community material theme  :  프로그램 배경 색을 추가해준다

material icon theme  : 작업중인 파일을 종류별로 아이콘으로 표시해준다.

만약 설치 후에도 적용이 안 된다면 파일 => 기본 설정 => 파일 아이콘 테마 => 메테리얼 선택

htmltagwrap : 선택 영역을 자동으로 html 태그로 묶는다.  드래그한 상태에서 알트+w를 누른다.

live server : 작업중인 파일이 자동으로 브라우저에서 업데이트 된다. 파일을 우클릭 한 후 오픈 위드 라이브 서버 클릭. 

open in browser : 원하는 브라우저로 파일을 열 수 있다. (이걸 깔면 단축키 Alt + b 로 바로 run 할 수 있다.)

auto rename tag : 한 쪽 태그만 수정해도 다른 쪽 태그가 자동으로 수정된다.

prettier : 파일 => 기본설정 => 설정 => 검색창에 Default Formatter 입력 => Prettier로 변경

=> 검색창에 Format On Save 입력 => 체크  => 세이브시 자동으로 문서를 교정해줌.

git extension pack : VSC 에서 실시간으로 문서를 깃허브로 저장할 수 있도록 한다. (리포지토리를 커밋하게 해준다.)

EJS language support : EJS 를 빠르게 타이핑 하도록 도와준다.

ES7+ React/Redux/React-Native snip : 리액트 문법을 도와 자동 작성하게 한다.

[ vsc 핵심 단축키 ]


파일 - 기본 설정 -  바로가기 키  (컨트롤 + S)   에 들어가면 모든 단축키 설정을 확인할 수 있다.

​

컨트롤 + D  :  원하는 단어를 선택한 후 컨트롤 D를 반복해서 누르면 위에서부터 같은 단어가 자동으로 선택된다. 원하는 만큼 선택했다면 한 번에 동시에수정할 수 있다.

알트 + 클릭   : 클릭하는 곳 마다 커서를 만들어준다.  동시에 원하는 코드를 입력할 수 있으며 입력을 끝냈다면 ESC를 눌러서 끝내면 된다.

알트 + 컨트롤 +  방향키  :  같은 열에 커서를 계속해서 만들어 나갈 수 있다.

알트 +  시프트 +  I   :  드래그 하여 선택한 코드의 마지막 칸에 커서를 생성한다. 

알트 + 시프트 + 드래그  :  마우스를 클릭하고 드래그하는 범위를 마우스 커서를 기준으로 수직 방향을 기준으로 자유롭게 정할 수 있다. 

알트 + 방향키 위, 아래  :   드래그 하여 선택한 코드를 하나의 묶음으로 삼아서 위 아래로 움직이게 해 준다.

알트 + 시프트 + 방향키 위, 아래   :  드래그 하여 선택한 코드를 하나의 묶음으로 삼아서 그대로 복사해준다.  빠르게 원하는 코드를 반복해서 복사할 때 유용하다.

코드를 드래그 한 후  컨트롤 +  k  +  f  : 자동 열 맞춤

컨트롤 + /   :  커서가 올려진 행에 있는 모든 코드를 코멘트 처리한다.

알트 + 시프트 + a : 드래그 하여 선택한 코드만 코멘트 처리한다.

컨트롤 +  home  /   컨트롤  + end   :  코드의 가장 윗 줄,  혹은 아래 줄로 커서를 이동시킨다.

home :  커서를 문장 가장 앞쪽으로 이동시킨다.

end : : 커서를 문장 가장 뒤쪽으로 이동시킨다.

컨트롤 + B  :  탐색기 사이드바를 없애거나 다시 킬 수 있다.

텝 :  오른쪽으로 열 맞춤

시프트 + 텝 : 왼쪽으로 열 맞춤

컨트롤 + a : 문서 전체를 선택한다.

시프트 + home or end : 커서로부터 왼쪽 혹은 오른쪽으로 한 문장이 끝나는 지점까지 선택한다.
